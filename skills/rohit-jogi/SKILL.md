## Major Technical Challenge (RAG Pipeline)

**Problem**: Configuration discovery across thousands of options in an enterprise logistics platform was slow and painful for users.

**Solution**: Built an AI chatbot with a RAG pipeline using LLMs, vector search, and function calling to let users query configuration in natural language.

**Tech**: Python, LLMs (various), vector search, HuggingFace embeddings, React frontend, FleetEnable API integration via function calling.

**Trade-offs**: Chose RAG over fine-tuning for flexibility — new configs don't require retraining. Function calling added complexity but enabled real-time API access to live data.

**Impact**: Drastically reduced time to find and configure options, improved UX for complex setup workflows.
